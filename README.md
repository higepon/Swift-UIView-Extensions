### How to use
Copy UIViewExtensions.swift to your project.

#### centerHorizontally(parentWidth: Double)
```swift
let title = UILabel(frame: CGRect(x: 50, y: 80, width: 200, height: 50))
title.text = "Hello, World"
title.sizeToFit()
title.centerHorizontally(320)
```
